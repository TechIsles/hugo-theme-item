## 快速开始

1. 安装 Hugo(`0.120.0`), go(`1.20.0`), Node.js(`20.17.0`)
2. 克隆仓库
   ```shell
    git clone https://github.com/fordes123/hugo-theme-item
    cd hugo-theme-item
    ```
3. 安装依赖
   ```shell
   npm install
   ```
4. 初始化
   ```shell
    npm run init
   ```
5. 调试运行
   ```shell
   npm run dev
   ```
